# delay

A basic delay with dry/wet control and cv controllable feedback and delay time.

##### Status: Work in progress

### controls

| Control | Function        |
|---------|-----------------|
| P1      | Dry Level       |
| P2      | Wet Level       |
| P3      | Feedback        |
| P4      | Delay Time    |
| CV1     | Feedback CV     |
| CV2     | Delay Time CV |
