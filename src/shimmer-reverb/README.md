# shimmer-reverb

An alteration to the basic reverb where you can add the wet level of the reverb signal one and two octaves up. This is also controllable via cv input 2.

### controls

| Control | Function        |
|---------|-----------------|
| P1      | Dry Level       |
| P2      | Wet Level       |
| P3      | Feedback        |
| P4      | Wet Octave Level |
| CV1     | Feedback CV     |
| CV2     | Wet Octave Level CV |
