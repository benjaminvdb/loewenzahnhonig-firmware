# reverb

A basic reverb with dry/wet control and cv controllable feedback and low pass frequency.

### controls

| Control | Function        |
|---------|-----------------|
| P1      | Dry Level       |
| P2      | Wet Level       |
| P3      | Feedback        |
| P4      | LP Frequency    |
| CV1     | Feedback CV     |
| CV2     | LP Frequency CV |
